# Making a Project Which Forecasts Average Stock Prices

**Python version:** 3.13.7  
**Author:** Lukasz Mamczura

## Technologies Used

The project uses the following libraries and frameworks:

- **yfinance** – for fetching stock market data  
- **pandas** – for data analysis and manipulation  
- **scikit-learn** – for machine learning models  
- **tensorflow** – for deep learning / neural networks  
- **fastapi** – for creating APIs (framework)  
- **uvicorn** – server for running FastAPI applications  
- **joblib** – for saving and loading models  
- **matplotlib** – for data visualization
